# ICT Systems

## Integration into existing networks

Integrating QKD systems into existing ICT systems is one of the biggest challenges and, at the same time, one of Q-net-Q's main goals. This includes connecting to routers, switches, firewalls, and network management systems. We develop solutions that enable the keys generated by QKD to be securely transferred to classic encryption protocols, thus making the entire communication chain quantum-secure. Compatibility with existing standards and protocols is a top priority in order to ensure a smooth transition.
